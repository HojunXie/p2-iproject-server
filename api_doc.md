# Movie Forum
Movie Forum is an application to discuss & talks about movies. This app has : 
* RESTful endpoint for asset's CRUD operation
* JSON formatted response

## Endpoint List
- GET /movies
- GET /movies/popular
- GET /threads/:id
- POST /threads
- DELETE /threads/:id
- GET /topics/:type/:id
- POST /topics
- DELETE /topics/:id
- POST /users/register
- POST /users/login
